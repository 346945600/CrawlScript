详细教程可看官方网站:http://crawlscript.github.io/

简单教程：

在CrawlScript-bin文件夹下可看到demo_baidu.js，这是一个实例，抽取百度所有的超链接信息，执行方式：

1.打开命令行，进入CrawlScript-bin目录。
2.执行： java -jar CrawlScript.jar demo_baidu.js。

执行完毕会发现终端显示了百度所有的超链接信息，同时，脚本将超链接文本写到了CrawlScript-bin文件夹下新建的log.txt中。

也可以直接打开CrawlScript的Shell来编写和执行CrawlScript,步骤：

1.打开命令行，进入CrawlScript-bin目录。
2.执行： java -jar CrawlScript.jar 进入CrawlScript的Shell。
3.开始编写CrawlScript,教程：http://crawlscript.github.io/
